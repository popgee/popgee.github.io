Windows 安装 tcping 命令以及使用方法

下载安装tcping.exe程序文件并安装放入到指定的系统目录中。

tcping和ping类似，不过tcping可以专门检查远程服务器的某个端口是否打开。

一、下载

首先下载tcping的exe程序https://elifulkerson.com/projects/tcping.php

二、安装

将下载后的tcping.exe文件移动到 C:\Windows\System32 目录下

三、使用方法
